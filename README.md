# Изменения в версии 1.2

## 1. Переименование вкладок
- **Было:** "Создать output.xlsx"
- **Стало:** "Создать справочник реквизитов (output.xlsx)"

*Причина:* Более понятное и информативное название для пользователей

## 2. Удаление функционала

### Удаленная вкладка
Полностью удалена вкладка **"Формирование шаблонов из таблицы"** вместе со всем связанным функционалом.

### Удаленные функции
- `process_files()` - функция обработки файлов и создания документов из таблицы
- Все связанные переменные и элементы интерфейса:
  - `output_file_var` - переменная для файла output.xlsx
  - `replace_from_var` - переменная для текста замены
  - `replace_to_var` - переменная для нового текста
  - `progress_bar2` - второй прогресс-бар
  - `status_label2` - второй статус-лейбл

### Причина удаления
Функционал дублировался с более продвинутым механизмом создания запросов во вкладке "Формирование запросов". Удаление упрощает интерфейс и устраняет путаницу.

## 3. Реорганизация вкладок

### Новая структура
1. **Вкладка 1:** "Создать справочник реквизитов (output.xlsx)" *(переименована)*
2. **Вкладка 2:** "Формирование запросов" *(бывшая вкладка 3)*
3. **Вкладка 3:** "Печать" *(бывшая вкладка 4)*

### Преимущества новой структуры
- Логичная последовательность действий:
  1. Создание справочника
  2. Формирование запросов на его основе
  3. Печать готовых документов
- Меньше вкладок - меньше путаницы
- Устранение дублирования функционала

## 4. Оптимизация кода

### Удаленные переменные из настроек
Из функций `save_settings()` и `load_settings()` удалены:
- `output_file` - больше не используется
- `replace_from` - больше не используется
- `replace_to` - больше не используется

### Обновленный список сохраняемых настроек
```json
{
    'working_folder': рабочая_папка,
    'compare_inn': флаг_поиска_ИНН,
    'compare_inn_value': ИНН_для_сравнения,
    'output_file_fz': путь_к_output.xlsx,
    'template_file_fz': путь_к_шаблону,
    'save_folder_fz': папка_сохранения,
    'print_folder': папка_печати,
    'start_number': начальный_номер
}
```

## 5. Визуальные изменения

### Обновленный интерфейс
- Перераспределение весов колонок для лучшего центрирования
- Сохранение единого стиля оформления
- Корректное отображение всех элементов после удаления вкладки

### Сохраненные элементы
Все остальные элементы интерфейса и функционал остались без изменений:
- Поиск и отбор организаций
- Создание документов с нумерацией
- Формирование текстовых файлов
- Печать документов
- Сохранение/загрузка настроек

## 6. Преимущества обновления

1. **Упрощение интерфейса** - меньше вкладок, меньше путаницы
2. **Логичная структура** - последовательность действий от создания справочника до печати
3. **Устранение дублирования** - единый механизм создания документов
4. **Чистота кода** - удален неиспользуемый код
5. **Понятные названия** - более информативные заголовки вкладок

## 7. Совместимость

### Обратная совместимость
- Старые файлы настроек `settings.json` будут корректно загружаться (неиспользуемые поля игнорируются)
- Все созданные ранее `output.xlsx` файлы полностью совместимы
- Сохранена возможность печати ранее созданных документов

### Рекомендации по обновлению
При переходе с предыдущей версии рекомендуется:
1. Создать резервную копию старых настроек (опционально)
2. При первом запуске настройки загрузятся автоматически
3. Проверить корректность путей в новом интерфейсе

# Коммерческие предложения в один клик v1.1

Программа для автоматизации создания запросов коммерческих предложений на основе шаблонов Word и данных из Excel.

## Основные возможности

### 1. Создание базы данных (output.xlsx)
- Извлечение информации из файлов `.doc` и `.docx`
- Автоматический парсинг: наименование организации, ИНН, адрес, email, телефон
- Поиск ИНН через Яндекс (опционально)
- Сохранение данных в структурированный Excel-файл

### 2. Формирование документов по шаблону
- Пакетная обработка организаций из Excel
- Автоматическая подстановка реквизитов в шаблон
- Замена произвольного текста в документе
- Сохранение результатов в папку "Итоговые_документы"

### 3. Формирование запросов с нумерацией
- Поиск организаций по наименованию или типу товара (нечеткий поиск)
- Выбор нескольких организаций для обработки
- Автоматическая нумерация запросов (№001, №002...)
- Создание документов с номерами в имени файла
- Генерация двух текстовых файлов:
  - `Адреса без номеров.txt` — список организаций
  - `Адреса запросов.txt` — организации с присвоенными номерами (№001 Название)
- Создание файла с реквизитами всех отобранных организаций

### 4. Печать документов
- Печать только первой страницы документов
- Печать всех страниц документа
- Массовая печать всех файлов из папки
- Отображение прогресса печати

### 5. Сохранение настроек
- Сохранение всех путей и параметров в JSON-файл
- Автозагрузка настроек при старте

## Требования
- Windows
- Microsoft Word (для печати)
- Python 3.6+ (если запуск из исходного кода)

## Формат входных данных
- **Документы для парсинга**: `.doc`, `.docx`
- **Шаблон документа**: `.docx`
- **Файл с данными**: `.xlsx` (создается программой)
